# 动漫 APW - Anime Preview Watch

## 项目介绍

本项目是一款面向动漫爱好者的「动漫预告片观看」网页应用，网页标题为「动漫 APW-在线 999」，其中 APW 为「Anime Preview Watch」的缩写，核心定位是提供动漫预告片的聚合浏览与观看能力。

## API 使用说明

1. 数据来源：本页面所有接口数据均来源于 [Jikan API](https://jikan.moe/)，该 API 为公开的动漫数据接口；

   ## 核心功能

5. 动漫预告片观看：支持大部分全球动漫预告片的在线播放（通过模态框嵌入 iframe 实现，无预告片时展示友好提示）；

6. 预告片搜索：支持关键词检索动漫预告片（关联导航栏搜索功能）；

7. 多维度筛选：支持按**年份、季节、动漫类型**筛选预告片，精准定位目标内容；

8. 分页浏览：提供页码跳转、上一页/下一页、指定页码跳转功能，支持大批量内容分页加载；

9. 响应式布局：适配不同设备尺寸（PC/平板/手机），优化不同屏幕下的展示体验；

10. 加载状态提示：数据请求过程中展示加载动画，提升用户体验；

11. 基础信息展示：展示动漫的日文标题、发行年份等基础信息，无数据时标注「未知」兜底。
    
    ## 技术栈
- 前端基础：HTML5

- 前端框架：Vue 2.x

- 网络请求：Axios

- UI 框架：Bootstrap 4.6.2（含配套 jQuery）

- 交互实现：Bootstrap Modal（模态框）、原生 JS/Vue 事件处理

- 布局适配：Bootstrap 栅格系统（响应式布局）
